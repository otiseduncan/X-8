# Speech And Voice

Default speech preference:

- Provider preference: Google
- Locale: `en-US`
- Gender: female
- Voice: `Google US English Female`

If Google Cloud TTS credentials are configured, XV8 reports Google Cloud TTS as configured. Without those credentials, the browser uses speech synthesis fallback and reports `browser_speech_synthesis`.

Speech input is visible in the cockpit as push-to-talk planning and transcript preview capability; browser support is reported honestly by the UI.
