# AI Team And Seed Knowledge

XV8 behaves like a senior product, development, design, QA, DevOps, security, and integration team by loading structured seed files from `knowledge/`.

Seed knowledge is not user memory. It is general operating knowledge used to improve planning, critique, and implementation.

## Team Seats

- Product Lead
- Senior Software Architect
- Backend Engineer
- Frontend Engineer
- UI/UX Designer
- AI Systems Engineer
- Prompt/RAG Engineer
- DevOps/Docker Engineer
- QA/Test Engineer
- Security Engineer
- Integration Engineer
- Technical Writer

Each seat includes responsibility, boundaries, reviews, common failure modes, and output style.
