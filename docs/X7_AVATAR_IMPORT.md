# X7 Avatar Import

Mount approved XV7/X7 files under `./imports/x7`.

XV8 scans likely avatar paths such as `shared/`, `assets/`, `public/avatar/`, `public/avatars/`, `data/assets/`, and `generated-assets/`.

Supported asset extensions include `.glb`, `.gltf`, `.vrm`, `.fbx`, `.obj`, `.usdz`, `.png`, `.jpg`, `.jpeg`, `.webp`, `.svg`, `.json`, `.anim`, and `.bin`.

Runtime import summaries are written to `runtime/import-reports/avatar-import-summary.md`.
